核心能力

九大引擎，构成安全左移的底座

从代码到运行态，从人工到智能，哨兵把主流应用安全技术统一编排到一条流水线上。

SAST 静态扫描

分析源代码中的安全漏洞：SQL 注入、XSS、密码硬编码、危险函数调用等。

SCA 组件分析

检测第三方依赖中的已知 CVE 漏洞与开源许可证风险。

DAST 动态扫描

对运行中的应用进行黑盒安全测试，发现运行态暴露面。

密钥泄露检测

扫描代码仓库中的敏感信息：API Key、密码、令牌、私钥。

AI 智能分析

自动分析漏洞根因、利用路径与修复方案，并能以对话方式答疑。

工单闭环

漏洞分配到人、SLA 跟踪、修复验证，全程可追溯。

CI/CD 集成

支持 GitLab CI、GitHub Actions、Jenkins 自动触发扫描。

安全门禁

基于漏洞严重度的构建阻断策略，问题不出流水线。

报告导出

安全摘要、SLA 报告、合规检查，支持 HTML / PDF 等 5 种格式。

项目管理与扫描

从登记项目到自动扫描

一个项目可登记多类信息，平台据此选择最合适的扫描方式，并在代码提交时自动触发。

项目管理 · 多维度登记
项目类型：Web 应用 / API 服务 / 移动端 / 基础设施
编程语言：Python / JavaScript / Java / Go / PHP / Ruby
目标 URL：用于 DAST 动态扫描（应用需处于运行状态）
仓库 / 本地路径：用于 SAST 静态分析与 SECRET 检测
操作：编辑、删除（级联清理关联扫描与漏洞）、一键扫描
扫描管理 · 触发与集成
新建扫描：选目标项目 → 选扫描类型 → 后台执行
CI/CD 集成：复制配置到 GitLab CI / GitHub Actions / Jenkins
定时扫描：周期性自动巡检，覆盖间歇暴露面
结果入库：扫描发现的漏洞自动进入漏洞管理
SAST

静态应用安全测试

分析源代码，定位注入、XSS、硬编码等代码层漏洞。

SCA

软件成分分析

扫描依赖树，匹配已知 CVE 与开源许可证风险。

DAST

动态应用安全测试

对运行中的应用黑盒测试，发现运行态暴露面。

SECRET

密钥泄露检测

扫描仓库中的 API Key、密码、令牌等敏感信息。

每一个漏洞，都有去处

02

03

04

关闭 Closed

智能分析双引擎

AI 既是分析师，也是顾问

哨兵内置两类 AI 能力：针对单条漏洞的深度分析，以及随时答疑的安全顾问对话。

功能一 · 漏洞 AI 分析
漏洞类型与严重性评估
攻击路径（详细步骤）
利用难度评估
业务影响分析
修复优先级建议

结果自动入库，刷新不丢失。支持 Ollama 本地模型或 OpenAI API。

功能二 · AI 安全顾问
这个 SQL 注入在生产环境危害有多大？怎么修？
哨兵安全顾问
结合当前漏洞上下文，建议如下：
  • 危害：可越权读取用户表，CVSS 约 9.1（Critical）
  • 修复：使用参数化查询 / ORM，禁止字符串拼接 SQL
  • 验证：修复后触发重新扫描确认闭环
